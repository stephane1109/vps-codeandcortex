Aide

Repères d’utilisation

Cette adaptation VPS conserve le moteur Rainette de la version Hugging Face, mais l’exécute dans une interface web locale au conteneur pour éviter la dépendance à Shiny côté utilisateur.

Les blocs latéraux couvrent maintenant les paramètres CHD, les choix spaCy, la classification simple ou double, le nettoyage, le NER, l’AFC et les cooccurrences.

L’onglet Explorateur reste le point d’entrée pour le graphe CHD interactif, tandis que Exports regroupe le ZIP global et les ZIP catégorisés.

Repères rapides

Utilisez classification simple pour une CHD classique. Passez en classification double si vous voulez rapprocher le comportement de rainette2() de la version source.

Activez le filtrage morphosyntaxique ou la lemmatisation spaCy seulement si vous souhaitez vraiment piloter la nature grammaticale des formes conservées.

Les sorties NER et AFC se remplissent uniquement si le calcul correspondant aboutit. En cas d’échec, le détail reste visible dans les logs et dans le statut des onglets concernés.
